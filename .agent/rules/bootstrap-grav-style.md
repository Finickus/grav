При редактировании файлов в этом проекте (особенно в папке user/pages/) соблюдай следующие правила:

Фреймворк: Используй только классы Bootstrap 5.3 для создания сеток и компонентов.

Сетка: Для структуры страниц всегда используй контейнеры .container, строки .row и колонки .col-md-*.

Сохранность YAML: Никогда не удаляй и не изменяй метаданные между разделителями --- в верхней части .md файлов, если об этом не попросили отдельно.

Формат контента: Если в файле указано markdown: false, пиши чистый HTML. Если markdown: true, используй шорткоды или HTML-блоки аккуратно, чтобы не сломать рендеринг Grav.

При каждом сохранении файла .md автоматически проверяй валидность Bootstrap-сетки и сообщай, если обнаружены лишние классы из других фреймворков

Для работы Изображений в Grav нужно использовать абсолютные пути, начиная с корня сайта.
