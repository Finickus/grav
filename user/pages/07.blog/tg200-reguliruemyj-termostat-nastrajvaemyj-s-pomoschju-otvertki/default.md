<p>TG200 — это **регулируемый термостат**, настройка которого производится **с помощью отвертки**.</p>
<h4 class="h5 mt-4">Ключевые особенности:</h4>
<ul class="ml-4">
<li>«Компенсированные» исполнения доступны по запросу.</li>
<li>Параметры контактов: **16(4) А 250В~** (высокая коммутационная способность).</li>
<li>Версии с чувствительным элементом, заполненным газом (TG200-Z), доступны по запросу.</li>
<li>Термостаты с **позолоченными контактами** доступны по запросу (для низковольтных или высокоточных применений).</li>
</ul>
