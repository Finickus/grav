<div class="card mb-5 shadow">
<div class="card-body">
<p class="fs-5">Котлы **Buderus (Будерус)** — это надежное немецкое оборудование, но даже оно может выдавать ошибки в работе. В этом подробном руководстве мы разберем все возможные коды ошибок, их причины и предоставим пошаговые инструкции по устранению неисправностей.</p>
</div>
</div>
<div class="card mb-5 shadow"><header class="card-header bg-info text-white">
<p class="card-h3 text-white fs-5"><span class="icon me-2"><i class="fas fa-info-circle"></i></span>Основные коды состояния котла</p>
</header>
<div class="card-body">
<div class="row g-3 content fs-6">
<div class="col-6">
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-broom"></i></span> **A** — Режим "трубочист"</li>
<li class="mb-1"><span class="icon me-2 text-danger"><i class="fas fa-fire"></i></span> **H** — Режим отопления</li>
<li class="mb-1"><span class="icon me-2 text-info"><i class="fas fa-shower"></i></span> **(=) H** — Режим ГВС</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-clock"></i></span> **0A** — Тактовая блокировка (ожидание)</li>
<li class="mb-1"><span class="icon me-2 text-success"><i class="fas fa-fire"></i></span> **0C** — Включение горелки</li>
</ul>
</div>
<div class="col-6">
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-thermometer-empty"></i></span> **0E** — Минимальная теплопроизводительность</li>
<li class="mb-1"><span class="icon me-2 has-text-secondary"><i class="fas fa-pause"></i></span> **0H** — Ожидание (нет потребности в тепле)</li>
<li class="mb-1"><span class="icon me-2 text-success"><i class="fas fa-play"></i></span> **0U** — Запуск котла</li>
<li class="mb-1"><span class="icon me-2 text-danger"><i class="fas fa-thermometer-full"></i></span> **0Y** — Температура подачи выше заданной</li>
</ul>
</div>
</div>
</div>

<div class="card mb-5 shadow"><header class="card-header bg-success text-white">
<p class="card-h3 text-white fs-5"><span class="icon me-2"><i class="fas fa-exclamation-triangle"></i></span>Коды ошибок и способы их устранения</p>
</header>
<div class="card-body content">
<div id="accordionBuderusErrors"><details open="" class="card mb-3 p-0 bg-light">
<summary class="h5 text-warning mb-0"><span class="icon me-2"><i class="fas fa-wind"></i></span> 2E - Функция удаления воздуха</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Решение:</p>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-sync"></i></span> Дайте котлу поработать в этом режиме.</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-wrench"></i></span> Проверьте автоматические воздухоотводчики.</li>
<li class="mb-1"><span class="icon me-2 text-info"><i class="fas fa-wind"></i></span> При необходимости вручную стравите воздух через краны Маевского.</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-danger mb-0"><span class="icon me-2"><i class="fas fa-fire"></i></span> 6A - Пламя не обнаружено</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Что проверить:</p>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-gas-pump"></i></span> Подачу газа (открыт ли кран).</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-tachometer-alt"></i></span> Давление газа (норма 13-20 мбар).</li>
<li class="mb-1"><span class="icon me-2 text-info"><i class="fas fa-bolt"></i></span> Состояние электродов розжига.</li>
<li class="mb-1"><span class="icon me-2 text-danger"><i class="fas fa-wind"></i></span> Тягу в дымоходе.</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-danger mb-0"><span class="icon me-2"><i class="fas fa-thermometer-full"></i></span> 4C - Срабатывание ограничителя температуры</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Причины и решения:</p>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-border-all"></i></span> Засорение теплообменника → промыть.</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-pump"></i></span> Неисправность циркуляционного насоса → проверить/заменить.</li>
<li class="mb-1"><span class="icon me-2 text-info"><i class="fas fa-wind"></i></span> Воздушные пробки → удалить воздух.</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-primary mb-0"><span class="icon me-2"><i class="fas fa-pump"></i></span> 2H - Защита насоса от заклинивания</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Что делать:</p>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-sync"></i></span> Проверить свободное вращение насоса.</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-broom"></i></span> Очистить крыльчатку от загрязнений.</li>
<li class="mb-1"><span class="icon me-2 text-info"><i class="fas fa-bolt"></i></span> Проверить напряжение питания насоса (220В).</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-warning mb-0"><span class="icon me-2"><i class="fas fa-thermometer-half"></i></span> 4U/4Y - Неисправность датчика температуры</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Диагностика:</p>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-ruler"></i></span> Проверить сопротивление датчика NTC (При 20°C ≈ 10-12 кОм).</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-plug"></i></span> Проверить целостность проводов.</li>
<li class="mb-1"><span class="icon me-2 text-danger"><i class="fas fa-exchange-alt"></i></span> При отклонениях — заменить датчик.</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-info mb-0"><span class="icon me-2"><i class="fas fa-fan"></i></span> 3A/3C/3L/3P/3Y - Ошибки вентилятора</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Пошаговая диагностика:</p>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-plug"></i></span> Проверить подключение и напряжение (220В).</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-sync"></i></span> Проверить свободное вращение крыльчатки.</li>
<li class="mb-1"><span class="icon me-2 text-danger"><i class="fas fa-exchange-alt"></i></span> При неисправности — заменить вентилятор.</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 has-text-secondary mb-0"><span class="icon me-2"><i class="fas fa-ellipsis-h"></i></span> Прочие редкие ошибки (7L/9L/EL, 8Y, 6C, 2P)</summary>
<div class="p-4 fs-6">
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-microchip"></i></span> **7L/9L/EL (Электроника):** Перезагрузить, проверить соединения, при повторе — заменить модуль KIM/регулятор BC20.</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-thermometer-half"></i></span> **8Y (Реле AT90):** Проверить подключение, при необходимости установить перемычку.</li>
<li class="mb-1"><span class="icon me-2 text-danger"><i class="fas fa-fire"></i></span> **6C (Ложное пламя):** Очистить электроды, проверить заземление, проверить герметичность газового клапана.</li>
</ul>
</div>
</details></div>
</div>

<div class="card-body content">
<div id="accordionCommonIssues"><details open="" class="card mb-3 p-0 bg-light">
<summary class="h5 text-primary mb-0"><span class="icon me-2"><i class="fas fa-power-off"></i></span> 1. Горелка не включается</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Проверьте:</p>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-toggle-on"></i></span> Положение аварийного выключателя.</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-bolt"></i></span> Состояние предохранителей.</li>
<li class="mb-1"><span class="icon me-2 text-danger"><i class="fas fa-wind"></i></span> Систему контроля дымовых газов.</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-warning mb-0"><span class="icon me-2"><i class="fas fa-volume-up"></i></span> 3. Посторонние шумы (бульканье)</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Причины:</p>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-border-all"></i></span> Накипь в теплообменнике → промыть.</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-wind"></i></span> Воздушные пробки → удалить воздух.</li>
<li class="mb-1"><span class="icon me-2 text-info"><i class="fas fa-tachometer-alt"></i></span> Низкое давление → долить теплоноситель.</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-danger mb-0"><span class="icon me-2"><i class="fas fa-power-off"></i></span> 4. Самопроизвольное отключение</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Диагностика:</p>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon me-2 text-primary"><i class="fas fa-wind"></i></span> Проверить тягу в дымоходе.</li>
<li class="mb-1"><span class="icon me-2 text-warning"><i class="fas fa-bolt"></i></span> Осмотреть ионизационный электрод.</li>
<li class="mb-1"><span class="icon me-2 text-info"><i class="fas fa-valve"></i></span> Проверить газовую арматуру.</li>
<li class="mb-1"><span class="icon me-2 text-danger"><i class="fas fa-thermometer-half"></i></span> Протестировать датчики температуры.</li>
</ul>
</div>
</details></div>
