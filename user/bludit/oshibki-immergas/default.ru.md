<p class="fs-5">Газовые котлы **Immergas** оснащены системой автоматической диагностики, которая значительно упрощает эксплуатацию. В случае неполадок система автоматически выводит соответствующий код ошибки на дисплей.</p>
<div class="notification is-warning mt-4">
<h5 class="h6 text-dark"><span class="icon me-2"><i class="fas fa-exclamation-triangle"></i></span>Важность понимания кодов ошибок:</h5>
<p class="mb-0 fs-6">Устранение сложных неисправностей (связанных с газом или электрикой) лучше доверить квалифицированному мастеру. Самовольный ремонт может привести к потере гарантии.</p>
</div>
</div>

<div class="card mb-5"><header class="card-header bg-info text-white">
<p class="card-h3 text-white fs-5"><span class="icon me-2"><i class="fas fa-cogs"></i></span>Общие рекомендации по устранению неполадок</p>
</header>
<div class="card-body">
<div class="row g-3">
<div class="col-lg-6 mb-3">
<div class="card h-100">
<h5 class="h6"><span class="icon me-2 text-primary"><i class="fas fa-sync"></i></span>Перезапуск котла</h5>
<p class="fs-6">При появлении кода ошибки, в первую очередь рекомендуется **перезапустить котел**, чтобы исключить системный сбой. Перезапуск обычно осуществляется нажатием кнопки **Reset** (около 5 секунд) или переключением питания.</p>
</div>
</div>
<div class="col-lg-6 mb-3">
<div class="card h-100">
<h5 class="h6"><span class="icon me-2 text-danger"><i class="fas fa-user-md"></i></span>Обращение к специалистам</h5>
<p class="fs-6">Если символы ошибки появляются снова после перезапуска, **строго рекомендуется обращаться в авторизованные сервисные центры Immergas**. Специалисты обладают квалификацией и оригинальными запчастями.</p>

<div class="card-body content"><details open="" class="card mb-3 p-0 bg-light">
<summary class="h5 text-primary mb-0"><span class="icon me-2"><i class="fas fa-fire"></i></span>1. Проблемы с розжигом пламени (01, E00, E02)</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Причины: Отсутствие пламени, низкое давление газа, неисправность электрода ионизации, неисправность платы.</p>
<h6 class="h6 mt-3 text-success"><span class="icon me-2"><i class="fas fa-wrench"></i></span>Решения:</h6>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon mr-1 text-info"><i class="fas fa-sync"></i></span> Перезапустить котел.</li>
<li class="mb-1"><span class="icon mr-1 text-warning"><i class="fas fa-gas-pump"></i></span> Проверить подачу газа (кран открыт, давление 20 мбар).</li>
<li class="mb-1"><span class="icon mr-1 text-primary"><i class="fas fa-broom"></i></span> Очистить электрод ионизации, отрегулировать зазор.</li>
<li class="mb-1"><span class="icon mr-1 text-success"><i class="fas fa-plug"></i></span> Проверить правильность подключения электропитания и заземления.</li>
<li class="mb-1"><span class="icon mr-1 text-danger"><i class="fas fa-magnet"></i></span> Проверить газовый клапан (катушки).</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-danger mb-0"><span class="icon me-2"><i class="fas fa-thermometer-full"></i></span>2. Перегрев котла (02, 12, E12, b19)</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Причины: Сигнал от термостата из-за перегрева, засорение каналов теплообменника накипью, нарушение теплообмена.</p>
<h6 class="h6 mt-3 text-success"><span class="icon me-2"><i class="fas fa-wrench"></i></span>Решения:</h6>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon mr-1 text-info"><i class="fas fa-snowflake"></i></span> Дать котлу остыть, затем снять блокировку.</li>
<li class="mb-1"><span class="icon mr-1 text-success"><i class="fas fa-broom"></i></span> **Выполнять чистку теплообменника** каждые пару отопительных сезонов.</li>
<li class="mb-1"><span class="icon mr-1 text-danger"><i class="fas fa-sync"></i></span> В худшем случае может потребоваться **замена сенсора**.</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-info mb-0"><span class="icon me-2"><i class="fas fa-wind"></i></span>3. Неисправность системы вентиляции (03, 11, E11)</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Причины: Недостаточная тяга, засорение дымохода, неисправность вентилятора или его блокировка.</p>
<h6 class="h6 mt-3 text-success"><span class="icon me-2"><i class="fas fa-wrench"></i></span>Решения:</h6>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon mr-1 text-warning"><i class="fas fa-broom"></i></span> Прочистить каналы систем вентиляции и дымоудаления.</li>
<li class="mb-1"><span class="icon mr-1 text-primary"><i class="fas fa-fan"></i></span> **Проверить вентилятор** (работоспособность, отсутствие засоров, напряжение).</li>
<li class="mb-1"><span class="icon mr-1 text-danger"><i class="fas fa-skull-crossbones"></i></span> При коде 03, немедленно **обратиться к специалистам** (риск угарного газа).</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-warning mb-0"><span class="icon me-2"><i class="fas fa-bolt"></i></span>4. Нестабильная работа электросети (37)</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Причины: Существенные скачки напряжения, снижение напряжения до критических значений (175-197 В).</p>
<h6 class="h6 mt-3 text-success"><span class="icon me-2"><i class="fas fa-wrench"></i></span>Решения:</h6>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon mr-1 text-info"><i class="fas fa-plug"></i></span> **Установить качественный стабилизатор напряжения** и/или ИБП.</li>
<li class="mb-1"><span class="icon mr-1 text-warning"><i class="fas fa-bolt"></i></span> Обеспечить надежное **заземление**.</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 text-success mb-0"><span class="icon me-2"><i class="fas fa-tint"></i></span>5. Недостаточная циркуляция воды (27, E18/E19)</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Причины: Нарушения в циркуляции, завоздушивание, закрытые краны, заклинивание насоса, загрязнение фильтров или теплообменника.</p>
<h6 class="h6 mt-3 text-success"><span class="icon me-2"><i class="fas fa-wrench"></i></span>Решения:</h6>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon mr-1 text-info"><i class="fas fa-wind"></i></span> Удалить воздух из системы.</li>
<li class="mb-1"><span class="icon mr-1 text-warning"><i class="fas fa-cogs"></i></span> Очистить/заменить циркуляционный насос, очистить фильтры.</li>
<li class="mb-1"><span class="icon mr-1 text-success"><i class="fas fa-door-open"></i></span> Убедиться, что все краны открыты.</li>
<li class="mb-1"><span class="icon mr-1 text-info"><i class="fas fa-tint"></i></span> При ошибках E18/E19 можно попробовать **добавить воду** в систему.</li>
</ul>
</div>
</details><details class="card mb-3 p-0 bg-light">
<summary class="h5 has-text-secondary mb-0"><span class="icon me-2"><i class="fas fa-tachometer-alt"></i></span>6. Разгерметизация системы / Низкое давление (10, 28, b26)</summary>
<div class="p-4 fs-6">
<p class="has-text-weight-bold">Причины: Утечка в контуре, низкое давление в отопительном контуре (ниже 0,9 бар).</p>
<h6 class="h6 mt-3 text-success"><span class="icon me-2"><i class="fas fa-wrench"></i></span>Решения:</h6>
<ul class="ml-4 list-unstyled">
<li class="mb-1"><span class="icon mr-1 text-primary"><i class="fas fa-search"></i></span> **Визуально осмотреть систему** на предмет протечек. Заменить потекшую деталь.</li>
<li class="mb-1"><span class="icon mr-1 text-success"><i class="fas fa-tachometer-alt"></i></span> **Восстановить давление** до 1,3 бар, используя кран подпитки.</li>
<li class="mb-1"><span class="icon mr-1 text-danger"><i class="fas fa-skull-crossbones"></i></span> При обнаружении запаха газа, немедленно **сообщить в горгаз**.</li>
</ul>
</div>
</details></div>

function searchError() {
    const searchTerm = document.getElementById('errorSearch').value.trim().toUpperCase();
    const resultDiv = document.getElementById('searchResult');
    
    if (!searchTerm) {
        resultDiv.innerHTML = '<div class="notification is-warning"><span class="icon me-2"><i class="fas fa-exclamation-triangle"></i></span>Пожалуйста, введите код ошибки для поиска.</div>';
        return;
    }
    
    const errorInfo = errorDatabase[searchTerm];
    
    if (errorInfo) {
        let solutionItems = '';
        // Используем более универсальное разделение решения
        const solutions = errorInfo.solution.split('. ').filter(s => s.trim().length > 0); 
        solutions.forEach(step => {
            solutionItems += `<li class="mb-2 fs-6"><span class="icon me-2 text-success"><i class="fas fa-check-circle"></i></span>${step.trim()}</li>`;
        });
        
        resultDiv.innerHTML = `
            <div class="card has-border-primary">
                <header class="card-header bg-primary text-white">
                    <p class="card-h3 text-white"><span class="icon me-2"><i class="fas fa-search"></i></span>Результаты поиска для ошибки <strong>${searchTerm}</strong></p>
                </header>
                <div class="card-body">
                    <h6 class="h6 mb-3 text-muted-dark"><span class="icon mr-1"><i class="fas fa-tag"></i></span> Категория: ${errorInfo.category}</h6>
                    <div class="content">
                        <h6 class="h6"><span class="icon mr-1"><i class="fas fa-file-alt"></i></span> Описание:</h6>
                        <p>${errorInfo.description}</p>
                        <h6 class="h6"><span class="icon mr-1"><i class="fas fa-wrench"></i></span> Решение:</h6>
                        <ul class="ml-4 list-unstyled">
                            ${solutionItems}
                        </ul>
                        <div class="notification is-info mt-4">
                            <h6 class="h6"><span class="icon me-2"><i class="fas fa-info-circle"></i></span>Информация:</h6>
                            <p class="mb-0 fs-6">Эта информация предоставлена для ознакомления. Для сложных неисправностей рекомендуется обратиться к квалифицированному специалисту.</p>
                        </div>
                    </div>
                </div>
            </div>
        `;
        // Прокрутить к результатам поиска
        document.getElementById('searchResult').scrollIntoView({ behavior: 'smooth' });
    } else {
        resultDiv.innerHTML = `
            <div class="notification is-danger">
                <h5 class="h6 text-white"><span class="icon me-2"><i class="fas fa-exclamation-circle"></i></span>Ошибка не найдена</h5>
                <p class="mb-0 fs-6">Информация по коду ошибки <strong>${searchTerm}</strong> в нашей базе отсутствует. Пожалуйста, сверьтесь с инструкцией к вашему котлу или обратитесь к специалисту.</p>
            </div>
        `;
    }
}

document.getElementById('errorSearch').addEventListener('keyup', function(event) {
    if (event.key === "Enter") {
        searchError();
    }
});
